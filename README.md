# MYTHORIA：神闻纪｜点击式邀请函

一个使用项目真实视觉素材制作的七幕沉浸式邀请函。访客点击画面、底部按钮，或使用键盘方向键逐幕推进，最后选择调查身份并被引导至微信公众号。

## 本地预览

直接打开 `index.html`，或运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 交互

- 点击画面或右箭头：下一面
- 左箭头：上一面
- 手机左右滑动：前后切换
- 第六面：选择身份后进入最终页面
- 最终页面：复制公众号名称并前往微信搜索

公众号名称目前配置为 `MYTHORIA 神闻纪`，可在 `script.js` 中修改。
